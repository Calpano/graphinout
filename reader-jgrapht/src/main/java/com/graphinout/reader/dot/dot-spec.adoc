== Mapping from DOT to CJ

|===
| DOT | CJ

| graph | ICjGraph
| node | ICjNode
| edge | ICjEdge
| attr_list | CjData of corresponding graph/node/edge
| port | ICjPort
| compass_pt | ICjPort.name
| subgraph | ICjGraph (nested)
|===
